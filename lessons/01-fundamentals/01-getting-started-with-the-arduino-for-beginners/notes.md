# Notes - Tutorial 1: Getting Started with the Arduino for Beginners

📝 Notes on the Code
setup() function: Runs once when the Arduino starts. Used to configure pin modes (input/output).

loop() function: Runs continuously after setup. This is where the main logic goes.

pinMode(pin, mode): Sets whether a pin is used for input or output.

digitalWrite(pin, value): Sends HIGH (5V) or LOW (0V) signal to a pin.

delay(ms): Pauses the program for the given time in milliseconds (1000 ms = 1 second).

Pin 13: On most Arduino boards, pin 13 is connected to the built-in LED.
