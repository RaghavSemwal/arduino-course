# Setup Notes

## Install Arduino IDE
1. Download from https://www.arduino.cc/en/software
2. Install the IDE and connect your Arduino Uno via USB.
3. In the IDE go to Tools > Board and select "Arduino Uno".
4. In the IDE go to Tools > Port and select the COM port / /dev/tty* your board shows up on.

## Test Upload
Open File > Examples > 01.Basics > Blink, click Upload, and confirm the onboard LED blinks.

## Recommended Kit
- Arduino Uno R3
- Breadboard + jumper wires
- Resistors: 220/330 ohm and 10k ohm assortment
- LEDs (assorted colors + RGB)
- Potentiometer
- Pushbuttons
- Buzzers (active + passive)
- Servo motor
- Stepper motor + ULN2003 driver
- DC motor + L298N/transistor driver
- 74HC595 shift register
- 16x2 LCD display
- DHT11 temperature/humidity sensor
- HC-SR04 ultrasonic distance sensor
- IR receiver + remote
